Finlite is a simple and human-friendly digital notebook that helps small business owners record sales and expenses quickly, see daily profit instantly and export a clean rreport at the day's end. 

## Core Features
FINLITE provides users with essential features such as:
 - Add sales (with timestamp)
 - Add expenses
 - Auto-calculate profit/loss
 - Daily summary
 - Offline mode for all inputs
 - Export to PDF
 - Simple dashboard
 - Phone number + PIN login
 - Basic data backup
