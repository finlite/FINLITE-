# FINLITE-
FINLITE is for people who don’t have time for complex apps. They want quick, simple, reliable money tracking.

## Core Features-
 • Add sales (with timestamp)
 • Add expenses
 • Auto-calculate profit/loss
 • Daily summary
 • Offline mode for all inputs
 • Export to PDF
 • Simple dashboard
 • Phone number + PIN login
 • Basic data backup

 ##Tech Stack-
 HTML
 Vanilla CSS
 Tailwind CSS
 JavaScript

MVP in development.
